# CJudge

**Online Judge for Tamkang University — ECE Programming Practice**

An independently built, non-profit online judge platform for ECE students at Tamkang University. Built and maintained by a TKU ECE student for self-study and peer practice.

## About

- 🎓 Built by a TKU ECE student, for TKU ECE students
- 📝 All problems are original, self-authored works
- 🆓 Non-profit, for educational use only
- 🌐 Publicly accessible at [command1000.github.io/cjudge](https://command1000.github.io/cjudge)
